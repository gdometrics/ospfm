This  file   allows  the   ``django-admin.py makemessages``   command  to  give
translation strings for currency names.

If a global currency (ie. a real currency, whose name is translatable) is added
to the  currencies list,  a new line  should be added to this file  in order to
make the currency's name translatable

This file may contain more currencies names than "initial_data.yaml".
